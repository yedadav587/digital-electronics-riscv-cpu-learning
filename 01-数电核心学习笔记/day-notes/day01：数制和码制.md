# 数电学习 Day01：数制与码制
## 学习日期
2026-04-03

## 学习内容
1.  常用数制：二进制、八进制、十进制、十六进制的表示与转换
2.  码制基础：8421 BCD码、余3码
3.  有符号数表示：原码、反码、补码
4.  补码的加减运算原理

## 核心知识点
1.  **进制转换**
    - 十进制 ↔ 二进制：整数部分除2取余，小数部分乘2取整
    - 二进制 ↔ 十六进制：每4位二进制对应1位十六进制
2.  **8421 BCD码**
    - 用4位二进制表示1位十进制数（0-9），是数字电路中最常用的编码方式
3.  **原码/反码/补码**
    - 原码：最高位为符号位，其余为数值位，0有两种表示
    - 反码：正数与原码相同，负数符号位不变，数值位按位取反
    - 补码：正数与原码相同，负数为反码+1，0有唯一表示，是CPU中实际使用的编码
4.  **补码运算**
    - 减法可转换为加法：A - B = A + (-B的补码)
    - 运算结果仍为补码，溢出时需要判断符号位

## 与CPU/芯片的关联
- 所有数字电路、CPU内部的运算，都以二进制为基础
- 补码是ALU实现加减运算的核心，是后续设计加法器、ALU的基础
- BCD码在数字显示、控制电路中应用广泛

## 学习心得
数制与码制是数电的基础，和计算机组成原理中数据表示的内容完全对应，理解起来很顺畅。补码的减法运算原理是今天的重点，明白了为什么CPU只用加法器就能实现减法。
